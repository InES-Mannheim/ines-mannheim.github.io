About SPA-CORE


The semantic process API (SPA) is a vocabulary and engine that enables managing semantic descriptions of process/data-models and its instances. SPA becomes the globally accepted academic standard and reference implementation. -- InES

Coming from the original vision about the SPA, the view slightly shifted towards more generality. While the main

focus for the first development phases are processes, the goal broadend to also support other domains. Thereby, three

Use Cases are the current implementation drivers; namely Semantic Process integration over format and tool borders,

Analytics like Process Mining and Process Optimization by combining usage data with process models.
